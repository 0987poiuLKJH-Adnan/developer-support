This sample shows how to create a unique value renderer to visualize a GeoEvent stream service.  

Please note that this sample will require you to install the ArcGIS GeoEvent Extentsion and publish your own stream service. Please refer to the following tutorial for additional help on setting this up.

Tutorial - Stream Services in GeoEvent:
http://www.arcgis.com/home/item.html?id=b087b8193b55465cb94d4c451dd541ac

In order to see the service, the Simulator must be running.
